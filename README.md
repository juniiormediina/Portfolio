# Proyecto: My Dev Portfolio

Este repositorio contiene el código fuente y recursos para mi portafolio personal de desarrollador frontend. La
estructura está pensada para crecer y adaptarse a futuro.

## Estructura del repositorio

- `frontend/` - Portafolio principal (React + TypeScript + Vite)
- `README.md` - Documentación general del repositorio
- `LICENSE` - Licencia del proyecto

## Frontend - Portafolio principal

- Interfaz moderna, responsiva y accesible
- Animaciones fluidas y componentes reutilizables
- Secciones: Inicio, Sobre mí, Proyectos, Habilidades, Contacto
- Integración con redes sociales y enlaces a proyectos
- Buenas prácticas de desarrollo y organización de código

> Consulta la documentación específica en la carpeta `Frontend` para más detalles e instrucciones de uso.
