# Proyecto: Dev Portfolio

Este proyecto es un portafolio de desarrollador frontend minimalista y responsivo, construido con React, TypeScript y
Vite. Incluye animaciones, componentes reutilizables y un diseño moderno basado en neumorfismo y Tailwind CSS.

## Características principales

- Interfaz moderna y responsiva
- Animaciones fluidas con motion/react
- Componentes reutilizables y organizados
- Secciones: Inicio, Sobre mí, Proyectos, Habilidades, Contacto
- Integración con redes sociales y enlaces a proyectos
- Accesibilidad y buenas prácticas de desarrollo

## Instalación y uso

1. Instala las dependencias:
   ```sh
   npm i
   ```

2. Inicia el servidor de desarrollo:
   ```sh
   npm run dev
   ```

3. Abre tu navegador en `http://localhost:5173` (o el puerto indicado en la terminal).

## Scripts disponibles

- `npm run dev`: Inicia el servidor de desarrollo
- `npm run build`: Genera la build de producción

## Estructura del proyecto

- `src/` - Código fuente principal
    - `components/` - Componentes reutilizables (Header, Hero, About, Projects, Skills, Contact, Footer)
    - `assets/` - Imágenes, datos y recursos
    - `styles/` - Estilos globales y utilidades
- `public/` - Archivos estáticos
- `index.html` - Punto de entrada HTML

## Tecnologías utilizadas

- React
- TypeScript
- Tailwind CSS
- motion/react
- Lucide React Icons
- Vite
